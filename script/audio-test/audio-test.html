NOTES: the following instructions are based on the button names and the i89 animation sequences.

  • When on load with 20deg F: Run Cold Night Intro. Hold establishing shot for 10 seconds.
  • When first zoomed into room: Run Play Heater Loop, fading to 1.0. When you back up to the "rest" position, run Play Heater Fading Out to 0.6 volume.
  • When intro animation is finished, Turn on HUD and run HUD Activate
  • When you zoom in to the window, run Play Heater Loop, fading to 1.0 AND Camera Zoom.
  • When you zoom out to the back position: run Play Heater Fading Out to 0.6 volume AND camera zoom.
  • when you pan to the left: run Play Heater Fading Out to 0.2 volume AND Play Cold Night Loop Fade In to 1.0
  • when you pan to the right: run Play Cold Night Loop Fade Out to 0.0 AND Play Heater Fading Out to 0.6 volume
  • when you move on to another scene: Turn off HUD and run HUD Deactivate AND run Fade All (when it works) AND run Unload All
